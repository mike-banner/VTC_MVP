# SKILL: UX/UI Elite Engineering

> Version: 2026.1 - Stealth Dark Focus

## DESCRIPTION

Ce skill transforme l'agent en architecte d'interface haut de gamme. Il remplace les composants "génériques" par des structures "Design System" basées sur le mode sombre, le glassmorphism et des échelles mathématiques.

## CAPABILITIES

- Génération de composants React/TS "Pixel Perfect".
- Audit d'accessibilité (WCAG 2.1) sur fond sombre (#050505).
- Implémentation de masques de saisie complexes (SIRET/TVA).
- Création de hiérarchie typographique fluide (Minor Third).

## ACTIVATION

Ce skill s'active dès qu'une tâche touche au Frontend, au Dashboard, ou à la création de formulaires.
